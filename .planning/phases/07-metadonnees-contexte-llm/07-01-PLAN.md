Enrichir les chunks FAISS avec le nom d'expérience source et labelliser le contexte injecté dans le LLM pour éliminer les mélanges de missions dans les réponses.

Output: `build_index.py` enrichi, `app.py` mis à jour, index FAISS rebuilté et commité.
</objective>
