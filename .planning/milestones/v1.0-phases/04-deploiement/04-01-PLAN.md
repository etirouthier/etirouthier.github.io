<task type="checkpoint:human-action">
  <name>Task 2: Déployer sur Streamlit Community Cloud et configurer le secret</name>
  <action>
    Déployer l'application sur Streamlit Community Cloud via le dashboard web (aucun CLI disponible pour cette étape).

    **Étapes exactes :**

    1. Aller sur https://share.streamlit.io
    2. Se connecter avec le compte GitHub associé au repo
    3. Cliquer "New app"
    4. Configurer :
       - Repository : `etienne.routhier` (ou le nom exact du repo GitHub)
       - Branch : `main`
       - Main file path : `app.py`
    5. Ouvrir "Advanced settings" AVANT de cliquer Deploy
    6. Dans la section "Secrets", ajouter :
       ```toml
       MISTRAL_API_KEY = "sk-..."
       ```
       (remplacer sk-... par la vraie clé API Mistral)
    7. Cliquer "Deploy!"
    8. Attendre que l'app démarre (peut prendre 2-5 minutes au premier déploiement)
    9. Copier l'URL de l'app déployée (format : https://{username}-{repo}-{hash}.streamlit.app)

    **Vérification post-déploiement :**
    - L'interface de chat est visible sans login
    - Taper une question (ex: "Quelles sont vos principales compétences ?") et vérifier qu'une réponse en français apparaît
    - Vérifier que la clé API n'apparaît pas dans le code source visible

    **Note :** Si Streamlit Cloud utilise `requirements.txt` par défaut, vérifier dans les settings de l'app qu'il pointe vers `requirements-app.txt`. Sinon, renommer `requirements-app.txt` en `requirements.txt` et déplacer l'ancien en `requirements-dev.txt`.
  </action>
